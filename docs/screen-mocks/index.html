How they rhyme. One finops.css design system: identical dark sidebar + filter bar, the same 4-up KPI strip, the same card + verdict pill + decision-bar pattern, a shared workload colour palette (Dataset blue · Dataflow violet · Power BI sky · Other slate) so colours mean the same thing on every screen, and the same honesty chips (✓ fixed / ⚠ not captured / ✗ gap). Each decision bar cross-links the other two, so the FinOps story reads as one narrative: capacity is over-subscribed → because Datasets dominate the mix → here's the bill and where it's wasted.
